# ScholarBird - Scholarship Finder App

A modern Flutter application for discovering and applying to scholarships tailored to your academic journey.

## Features

- 🔍 **Discover Scholarships** - Find scholarships tailored for your academic journey
- 📊 **Track Applications** - Keep track of your application progress in one place
- 🎯 **Smart Filtering** - Easily filter scholarships by field, country, and degree
- 💾 **Save Favorites** - Bookmark scholarships for later review
- 👤 **User Profile** - Manage your profile and preferences

## Getting Started

### Prerequisites

- Flutter SDK (>=3.10.0)
- Dart SDK (>=3.0.0)

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/ScholarBird.git
cd ScholarBird
```

2. Install dependencies
```bash
flutter pub get
```

3. Run the app
```bash
flutter run
```
